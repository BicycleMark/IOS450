Before you begin

This is an iOS Objective-C bindings class. To complete the coding exercises, you will need a iOS development environment as well as direct access to Xcode with either a simulator or a device to run the exercise on. Please see the Windows setup or macOS setup pages if you need help installing the Xamarin development environment.

Overview

A Bindings Library is an assembly containing Managed Callable Wrappers for Objective-C types. This course introduces the core concepts in terms of Libraries, Frameworks and creating C# wrapper classes for Objective-C libraries. The content is a prerequisite to using the complete Xamarin.iOS tool chain used to create iOS Binding libraries.
